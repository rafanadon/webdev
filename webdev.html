These are the steps to push a github repository

First lets discuss what is git?
git is a free and open source distributed control system design to handle
everything from small to very large project with speed and efficiency

Here are the steps to push your code to your own repository.


            1. Create a folder specifically for your project
            # mkdir project 1

            2. use git init

            3. create a html file by using 
            echo "" > README.md

            4. To add a contents of the file README.md
            use  git add  
            git add README.md

            5. To commit the file
            git commit -m "FIRST COMMIT"

            6. Create branch 
            git branch -M main 

            7. To view the status branch and view if there is untracked files
            git status

            8. to select the local repository and connect this to the github repository use 
            git remote add origin https://github.com/your-repository/your-username/webdev-raf.git
            
            9. to push the repository
            git push -u origin main 

            10. Git bash will need your email or user name 
            use
            git config --global user.email baring your user email. 

            11. Your done! Now, you can see your repository in github.

These are the steps to push a github repository

First lets discuss what is git?
git is a free and open source distributed control system design to handle
everything from small to very large project with speed and efficiency

Here are the steps to push your code to your own repository.


            1. Create a folder specifically for your project
            # mkdir project 1

            2. in the gitbash you need to go the the folder you created by using cd 
            cd project1

            3. use git init

            4. create a html file by using 
            echo "" > README.md

            5. To add a contents of the file README.md
            use  git add  
            git add README.md

            6. To commit the file
            git commit -m "FIRST COMMIT"

            7. Create branch 
            git branch -M main 

            8. To view the status branch and view if there is untracked files
            git status


            9. to select the local repository and connect this to the github repository use 
            git remote add origin [your repository link / URL]
            
            10. to push the repository
            git push -u origin main 

            11. Git bash will need your email or user name 
            use
            git config --global user.email baring [your user email]. 

            12. Your done! Now, you can see your repository in github.

            13. Create your own Html files by using the git bash
            use
            echo "" > [yourfile].html 

            14. Code your html file in VS code 
            
            15. After you code your html file and want to put it in your github repository you need to pull first 

            16. To pull your repository
            git pull [your repository link]. 

            Output 

                     git pull https://github.com/rafanadon/webdev
                    From https://github.com/rafanadon/webdev
                    * branch            HEAD       -> FETCH_HEAD
                    Already up to date.
                    

            17. After pulling the the repository you need to add the html file by using 
            git add [yourfilename].html

            18. git status to view the gitbash status

            Output
                    git status
                    On branch main
                    Your branch is up to date with 'origin/main'.
                    
                    Changes to be committed:
                    (use "git restore --staged ..." to unstage)
                            modified:   webdev.html
            

            19. git commit -m "[your commit message]"
            
            Output 

                    $ git commit -m "Updated"
                    [main 11d0ed0] Updated
                    1 file changed, 22 insertions(+), 2 deletions(-)
                    

            
            20. git push -u

            Output
                    git push -u
                    Enumerating objects: 5, done.
                    Counting objects: 100% (5/5), done.
                    Delta compression using up to 4 threads
                    Compressing objects: 100% (4/4), done.
                    Writing objects: 100% (4/4), 1.48 KiB | 1.48 MiB/s, done.
                    Total 4 (delta 0), reused 0 (delta 0), pack-reused 0
                    To https://github.com/rafanadon/webdev.git
                    83eebf9..cccac53  main -> main
                    branch 'main' set up to track 'origin/main'.
            
            
            
            21. Congratulations! the html file is in the git hub repository now.
